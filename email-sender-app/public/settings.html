Update Email Settings

Instructions to set up Gmail OAuth2 for sending emails:
  1. Go to the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Enable the Gmail API for your project.
  4. Go to "Credentials" and create an OAuth 2.0 Client ID (Web application).
  5. Set authorized redirect URIs to: https://developers.google.com/oauthplayground
  6. Copy the Client ID and Client Secret from the credentials.
  7. Go to OAuth 2.0 Playground.
  8. Click the gear icon, check "Use your own OAuth credentials", and enter your Client ID and Secret.
  9. Select "https://mail.google.com/" from the list and authorize access.
  10. Exchange the authorization code for tokens, and copy the Refresh Token.
  11. Enter your Gmail address, Client ID, Client Secret, and Refresh Token below.

Note: Each user must configure their own Gmail OAuth2 credentials. The app uses these to send emails from your Gmail account.
